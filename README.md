# Food Hub - Android Application

This is the Android version of the Food Hub application, built with modern Android development techniques.

## Project Overview

This project follows the MVVM (Model-View-ViewModel) architecture pattern, combined with Clean Architecture principles to create a scalable, maintainable, and testable application.

### Tech Stack

- **UI:** Jetpack Compose with Material 3
- **Asynchronous Programming:** Kotlin Coroutines and Flow
- **Dependency Injection:** Hilt
- **Database:** Room for local caching
- **Networking:** Retrofit for API communication
- **Animations:** Lottie for rich, engaging animations
- **Push Notifications:** Firebase Cloud Messaging
- **Location:** Google Play Services for location tracking
- **Authentication:** Biometric authentication
- **On-device AI:** Gemini Nano for personalized recommendations

## Getting Started

To get started with this project, you'll need Android Studio and a device or emulator. Once you have the project cloned, you can build and run the app from Android Studio.
